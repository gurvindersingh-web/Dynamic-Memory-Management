2 Page Replacement Simulation Flow

The internal logic when a page reference is processed: checking cache hit/miss, selecting a victim page for eviction based on the chosen algorithm (FIFO/LRU/Optimal), updating the TLB and page table, computing the working set, and detecting thrashing conditions.

3 Dynamic Partitioning Flow

How the partitioning simulator handles memory allocation using four placement algorithms (First Fit, Best Fit, Worst Fit, Next Fit), block splitting when a hole is larger than needed, deallocation with adjacent free block coalescing, and fragmentation metric calculation.

flowchart TD A["📥 Process Arrives\n(size, duration)"] --> B{"Event Type?"} B -->|Allocation| C{"Which Algorithm?"} C -->|First Fit| D["Scan from beginning\nPick FIRST block ≥ size"] C -->|Best Fit| E["Scan ALL blocks\nPick SMALLEST ≥ size"] C -->|Worst Fit| F["Scan ALL blocks\nPick LARGEST ≥ size"] C -->|Next Fit| G["Scan from last pointer\nPick FIRST ≥ size"] D --> H{"Block Found?"} E --> H F --> H G --> H H -->|Yes| I["Allocate Process"] I --> J{"Block > Process?"} J -->|Yes| K["Split Block\nCreate free remainder"] J -->|No| L["Exact Fit"] K --> M["📊 Update Memory Map"] L --> M H -->|No| N["❌ Allocation Failed\nExternal Fragmentation"] N --> M B -->|Deallocation| O["Free Process Block"] O --> P{"Adjacent Free?"} P -->|Yes| Q["Coalesce Blocks"] P -->|No| R["Mark as Free"] Q --> M R --> M M --> S["Calculate Fragmentation"] S --> T["Record Snapshot"] T --> U{"More Events?"} U -->|Yes| A U -->|No| V["🏁 Complete"]

4 Three-Panel UI Architecture

The main simulator interface is organized into three panels: the Config Panel (left) for algorithm selection and playback controls, the Simulation Stage (center) for real-time memory visualization, and the Stats Panel (right) for analytics, charts, and event logging.

5 Theme System Flow

The theme system supports 9 themes (6 dark + 3 light). When a user selects a theme, CSS custom properties are injected into the document root. A MutationObserver detects the change and triggers Recharts to re-read the computed color values for SVG rendering.
